🔧 Configuration EmailJS 📖 Créer un compte gratuit
📋 Guide rapide EmailJS :
1. Créez un compte sur emailjs.com
2. Ajoutez un service email (Gmail, Outlook…) → notez le Service ID
3. Créez un template avec les variables : {{ref}}, {{stock}}, {{seuil}}, {{user}}
4. Copiez le Template ID et la Public Key (dans Account → API Keys)
5. Collez les 3 valeurs ci-dessus et sauvegardez
